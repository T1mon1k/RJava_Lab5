## Опис проекту
Цей проєкт реалізує таксопарк на Java та демонструє ієрархію легкових автомобілів. Є базовий клас Car 
і класи-нащадки EconomyCar, BusinessCar, PremiumCar та SuperCar. Таксопарк працює з масивом автомобілів, 
вміє підраховувати загальну вартість автопарку, сортувати авто за витратами пального, знаходити машини у 
заданому діапазоні швидкості та виводити інформацію у табличному форматі. Усі вхідні дані проходять 
перевірку, а можливі помилки обробляються через винятки.
